# Les Amour

**"Rediscover the Art of Us."**

Les Amour is a premium, marriage-first intimacy and relationship platform designed specifically for married Christian couples (aged 28-45). It provides a sanctuary for couples to reconnect through AI coaching, intentional courses, a supportive community, and a curated boutique.

## Platform Features

- **AI Relationship Coach:** Warm, wise guidance rooted in emotional intelligence and faith principles (GPT-4o powered).
- **Course Library:** Deep-dive modules on touch, communication, and sacred intimacy.
- **Covenant Club (Membership):** Weekly challenges and monthly digital date night kits.
- **The Boutique:** Curated affiliate store featuring connection tools and intimacy essentials.
- **Common Hearth Forum:** An anonymous, moderated community for couples.

## Tech Stack

- **Frontend:** HTML5, CSS3 (Design System), JavaScript (ES6+).
- **Backend/Database:** Supabase (PostgreSQL, Auth, Real-time).
- **AI Engine:** OpenAI GPT-4o API.
- **Payments:** PayPal, Crypto (ERC-20), Payoneer (No Stripe).

## Design Identity

- **Colors:** Deep Crimson (#8B0037), Champagne Gold (#D4AF37), Blush (#FAF0F0).
- **Typography:** Playfair Display (Headings), Inter (Body).
- **Tone:** Luxury, sacred, intimate, warm, and covenant-rooted.

## Directory Structure

- `/landing`: Marketing landing page.
- `/app`: Member dashboard and AI Coach interface.
- `/courses`: Course library and lesson player.
- `/store`: Curated boutique (affiliate links).
- `/community`: Anonymous forum.
- `/membership`: Covenant Club details and sign-up.
- `/shared`: Global CSS design system and JavaScript logic.
- `/data`: JSON stores for challenges and date kits.

## Deployment

1. Configure `shared/app.js` with your Supabase and OpenAI credentials.
2. Execute `schema.sql` in your Supabase SQL Editor.
3. Host the files on any static hosting provider (Netlify, Vercel, GitHub Pages).
