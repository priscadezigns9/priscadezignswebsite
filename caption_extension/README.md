# Priscion Caption AI

**"High-Fidelity Captions. Sovereign Speed."**

Priscion Caption AI is a Chrome Extension built for the Priscion Empire. It allows creators to generate professional, AI-powered social media captions for any image on the web with a simple right-click.

## Features

- **Context Menu Integration:** Right-click any image and select "Generate Caption with Priscion AI".
- **Multi-Platform Support:** Instantly get caption variations for Instagram, Threads, Facebook, and Twitter/X.
- **Tone Selection:** Choose from Professional, Hype, Inspirational, Luxury, or Spiritual (TWMK).
- **AI-Powered:** Uses OpenAI GPT-4o-mini for unique, high-quality captions.
- **Template Fallback:** Over 30+ built-in templates ensure you always have a great caption, even without an API key.
- **Smart Hashtags:** Automatically includes 20+ hashtags for IG and 5 for Twitter.
- **Character Counter:** Real-time character limits for each platform.
- **History:** Keeps track of your last 10 generated captions.
- **Custom Branding:** Append your own Shop URL to every post.

## Installation (Developer Mode)

1. Download or clone this repository.
2. Open Chrome and navigate to `chrome://extensions/`.
3. Enable **Developer mode** (toggle in the top right).
4. Click **Load unpacked** and select the `caption_extension` folder.

## Configuration

1. Click the extension icon and select **Settings** (gear icon) or go to Options.
2. Enter your **OpenAI API Key** for personalized AI captions.
3. Set your **Shop URL** to automatically link your products.
