# Priscion Price Scout

"Never Overpay. Sovereign Savings."

Priscion Price Scout is a Manifest V3 Chrome Extension designed for the Priscion Empire. It helps users track prices, find coupons, and discover alternative high-fidelity products across the empire's ecosystem.

## Features
- **Affiliate Integration**: Automatically injects `priscadezigns-20` into Amazon product URLs.
- **Price History**: Visualizes historical price trends via CamelCamelCamel.
- **Coupon Scout**: Detects checkout pages and auto-suggests valid coupons.
- **Empire Mapping**: Direct links to relevant Priscion shops (Tech, Shoes, Fitness, etc.) based on the product category currently being viewed.
- **Deal Score**: 1-10 rating on the current value.
- **Price Alerts**: Background notifications for price drops.

## Installation
1. Download this repository.
2. Open Chrome and go to `chrome://extensions/`.
3. Enable "Developer mode".
4. Click "Load unpacked" and select the `pricescout_extension` folder.
