Professional Profile

Sunil is an AI IT Support Officer deployed by Prisca Dezigns for Caribbean businesses. He operates as the first-response layer for all technology issues — receiving support requests via WhatsApp and email, logging structured tickets, resolving common problems autonomously, and escalating anything beyond his threshold so no issue goes untracked. Sunil was purpose-built for the Caribbean SME reality: minimal dedicated IT staff, heavy WhatsApp dependency, and systems that can't afford unmonitored downtime. At Enterprise tier, he expands into 24/7 system surveillance, data integrity auditing, and after-hours critical incident management.

Core Responsibilities

  • Receive IT support requests via WhatsApp and email — assign ticket numbers and acknowledge immediately
  • Handle common Tier 1 requests autonomously: password resets, account unlocks, software access provisioning
  • Coordinate new user onboarding — accounts, credentials, and full access provisioning checklist
  • Log every request in a structured IT ticket register with status tracking and timestamps
  • Full conversational Help Desk Agent — step-by-step WhatsApp troubleshooting without IT staff involvement
  • IT asset register — hardware, software, and assigned users tracked and maintained
  • Licence and subscription renewal alerts — flagged 30 days before expiry, zero missed renewals
  • Weekly cybersecurity awareness broadcasts — phishing alerts and security notices to all staff via WhatsApp
  • Perform scheduled data integrity checks — detect duplicates, missing fields, and format violations
  • 24/7 system uptime surveillance — real-time monitoring of latency, downtime, and platform anomalies
  • After-hours critical incident escalation — no P1 issue waits until morning
  • Monthly IT activity report and bi-weekly data integrity report delivered on schedule
