# D'Panyard - Trinidad Business Directory

A standalone Web2 platform connecting Trinidadian businesses and tradespeople with clients. 

## Design Philosophy: "The Yard"
"The Yard" is the signature UI component of D'Panyard. Instead of a sterile, generic list of businesses, "The Yard" presents businesses as a digital street. Each business card is styled like a shop front, emphasizing the local, neighborly feel of a Port of Spain community.

## Tech Stack
- Frontend: HTML5, CSS3 (Design Tokens), JavaScript (Vanilla)
- Styling: Custom CSS with a palette of Steel Silver, Deep Navy, and Warm Amber.
- Integration: Direct WhatsApp (wa.me) integration for lead generation.

## Folder Structure
- `/landing`: Home page with hero, search, and "The Yard" preview.
- `/browse`: Directory search and filtering with interactive "Yard" grid.
- `/business`: Business owner dashboard with analytics and lead tracking.
- `/shared`: Common design tokens and base styles.

## Deployment
Built for regional expansion across the Caribbean. Currently optimized for Trinidad & Tobago.
