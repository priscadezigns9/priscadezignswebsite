# Calalloo Marketplace

Calalloo is a standalone Web2 marketplace dedicated to everything Caribbean. It connects Caribbean vendors, artisans, and creators with the diaspora and local consumers.

## Design Philosophy
- **Warm & Alive**: Inspired by the feeling of a Caribbean market on a Sunday morning.
- **Vibrant Palette**: Deep saffron, burnt ochre, forest green, and midnight indigo.
- **The Island Selector**: A custom interactive SVG map that serves as the primary discovery tool.
- **Authentic Voice**: Using warm, local expressions like "Aye, look what we have."

## Key Components
- **The Island Selector**: Implemented as a responsive SVG with interactive paths for Trinidad, Jamaica, Barbados, Guyana, St Lucia, Grenada, Antigua, Bahamas, Haiti, and the Dominican Republic.
- **Masonry Browse**: A product grid that feels like a real market stall layout, avoiding the uniform "cold" look of traditional e-commerce.
- **Seller Dashboard**: A simplified, high-fidelity interface for Caribbean makers to manage their listings and track earnings.

## File Structure
- `landing/index.html`: Full marketing experience.
- `buyer/browse.html`: Product discovery with the Island Selector and filters.
- `seller/dashboard.html`: Vendor management portal.
- `shared/calalloo.css`: Design tokens and global styles.

## Tech Stack
- Pure HTML5, CSS3, and Vanilla JavaScript.
- Custom SVG for interactive mapping.
- Inter font family for high-fidelity UI.
