AI Employee Compliance Mandate

Legal ComplianceAll AI employees operate within the laws of Trinidad & Tobago and CARICOM regional standards.

Fully AutonomousEvery employee executes their role without requiring human intervention. Humans enhance — AI executes.

Repetitive Tasks OnlyAI employees handle high-frequency, repeatable tasks. One-off or rare tasks remain human-led.

Ethical by DesignAll actions must be transparent, fair, and non-deceptive. No AI employee misrepresents itself as human.

Fully AuditableEvery action is logged, traceable, and reviewable. No silent edits. Every employee leaves a trace.

Human + AI PartnershipAI employees empower your human team — they do not replace human judgement, creativity, or leadership.

Platform Infrastructure

Shared Infrastructure Tools

The backbone all AI employees run on — self-hosted, open-source, and built to scale.

Hosting Layer
Coolify
Self-hosted deployment platform on a VPS (Hetzner / DigitalOcean). Hosts n8n, Supabase, Open WebUI, and backend services — replacing per-service SaaS costs with one flat server fee (~$6–$10/mo).
AI Dev Agent
OpenHands
Open-source autonomous AI software agent. Writes code, runs it, fixes bugs, and executes terminal commands without manual intervention. Builds and maintains backend logic, APIs, and automation scripts for the full AI employee stack.
Web Data Extraction
Maxun
No-code web scraping platform. Point, click, and train a robot to extract structured data from any website automatically. Powers Dean's competitor research, lead intelligence, and market signal feeds into Supabase.
AI Interface Layer
Open WebUI
Self-hosted ChatGPT-style interface connecting to any AI model (Claude, GPT-4o, Ollama). Serves as the internal AI control panel and the foundation UI layer for the Orcinos platform. Runs fully offline if needed.
Web Agent / Browser Automation
Browser Use
AI agent that controls a real browser — opens pages, clicks, fills forms, and completes tasks like a human. Gives Sierra, Dean, and Alana hands on the web for form submissions, research, and admin tasks across any portal.
AI Workflow Builder
Langflow
Low-code drag-and-drop builder for AI agents and RAG pipelines. Connect LLMs, vector databases, memory, and tools visually — no code required. Used to build and deploy the logic flows that power each AI employee's brain and decision-making.

Orcinos

Unified Platform Stack

All tools combine to form the backbone of Orcinos — a sovereign AI platform with Caribbean Contextual Intelligence and Neural Connectors (WhatsApp, Outlook, Google Calendar).
