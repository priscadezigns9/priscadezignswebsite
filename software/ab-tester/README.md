# SPLITLAB - No-Code Landing Page A/B Tester

SplitLab is a professional Web2 SaaS platform designed to help marketers and founders optimize their conversion rates without touching a line of code.

## Key Features
- **AI-Powered Variants:** Automatically suggests headlines, CTAs, and layout changes.
- **Visual Traffic Routing:** Split users across up to 3 variants with 0ms flicker.
- **Statistical Significance:** Built-in engine ensures data accuracy before declaring a winner.
- **Heatmaps:** Integrated visual click tracking per variant.

## Tech Stack
- Frontend: HTML5, Tailwind CSS, Vanilla JS
- Deployment: Static Hosting ready
- Backend Interface: RESTful AI Integration

## Folder Structure
- `/landing`: Public-facing marketing page.
- `/app`: User dashboard and test management system.
