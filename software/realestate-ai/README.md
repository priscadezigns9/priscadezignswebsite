# LISTCRAFT - AI Real Estate Listing Generator

ListCraft is a Web2 SaaS built for real estate agents and brokerages to streamline their marketing operations using generative AI.

## Core Features
- **Listing Package Generation:** One input → MLS copy, social media posts, email templates, and video scripts.
- **Tone Customization:** Switch between Luxury, Family, Investment, and more to match the property vibe.
- **Media Ready:** Designed to handle high-resolution property photography and media metadata.
- **History Tracking:** Manage and regenerate past listings with updated pricing or features.

## Technical Architecture
- Mobile-first, responsive dashboard.
- High-fidelity visual language (Violet accent).
- Static-export ready for easy integration into existing realtor workflows.
