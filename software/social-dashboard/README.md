# SOCIAL_AI - AI Social Media Dashboard

The ultimate command center for your social media presence, powered by AI.

## Features
- **Unified Scheduler:** Plan and schedule content across IG, Threads, FB, X, LinkedIn, and TikTok.
- **AI Content Generator:** Transform a single topic into optimized posts for every platform.
- **Unified Inbox:** Manage all your DMs and comments in one place (Coming Soon).
- **Smart Analytics:** Understand your reach and engagement with cross-platform data.

## Tech Stack
- Frontend: HTML5, CSS3 (Inter font), Vanilla JavaScript
- Theme: Dark (#000) with Coral Red (#FF6B6B) accents

## Getting Started
1. Open `landing/index.html` for the marketing overview.
2. Open `app/dashboard.html` to explore the unified dashboard UI.
