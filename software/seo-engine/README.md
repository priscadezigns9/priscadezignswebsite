# SEO_ENGINE - AI SEO Content Engine

A comprehensive SEO strategy and content generation platform powered by AI.

## Features
- **Page Audit:** Detailed technical and structural analysis of any URL.
- **Content Generator:** Create SEO-optimized blog posts from target keywords.
- **Content Calendar:** Plan your 30-day content strategy automatically.
- **Meta Rewriter:** Optimize your titles and descriptions for maximum CTR.

## Tech Stack
- Frontend: HTML5, CSS3 (Inter font), Vanilla JavaScript
- Theme: Dark (#000) with Emerald Green (#00C896) accents

## Getting Started
1. Open `landing/index.html` to view the marketing page.
2. Open `app/dashboard.html` to access the application UI.
