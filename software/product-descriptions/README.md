# DESCAI - AI Product Description Generator

Transform your e-commerce catalog with AI-powered sales copy and SEO optimization.

## Features
- **Bulk Rewrite:** Update hundreds of product descriptions in one click.
- **Tone Selector:** Choose from Professional, Conversational, Luxury, Technical, or Fun styles.
- **Platform Integrations:** Direct sync with Shopify, WooCommerce, Etsy, and BigCommerce.
- **Image Optimization:** Automatic SEO-friendly alt text generation for product images.
- **SEO Scoring:** Real-time feedback on keyword density and readability.

## Tech Stack
- Frontend: HTML5, CSS3 (Inter font), Vanilla JavaScript
- Theme: Dark (#000) with Orange (#FF9500) accents

## Getting Started
1. View the landing page at `landing/index.html`.
2. Access the catalog management interface at `app/dashboard.html`.
