# PawDx - AI-Powered Pet Health Platform

PawDx is an Orcinos product designed to provide pet parents with instant peace of mind. Using AI-driven symptom analysis, a comprehensive breed health library, and integrated health tracking, PawDx bridges the gap between pet owners and professional veterinary care.

## 🐾 Core Features

- **AI Symptom Checker**: Real-time analysis of symptoms (GPT-4o-mini).
- **Urgency Meter**: Color-coded urgency levels (Green/Yellow/Red) for immediate action guidance.
- **Breed Library**: Database of 200+ breeds with common health predispositions.
- **Vaccine Tracker**: Log and track pet vaccinations with next-due dates.
- **Health Log**: Keep a detailed diary of your pet's wellness and export logs for vet visits.
- **Pet Profiles**: Manage multiple pet profiles in a single account.

## 🛠 Tech Stack

- **Frontend**: HTML5, CSS3 (Vanilla), JavaScript (Vanilla)
- **Backend**: Supabase (Authentication & PostgreSQL Database)
- **AI Engine**: OpenAI GPT-4o-mini API
- **Design**: Mobile-first, friendly, and trustworthy aesthetics.

## 🚀 Setup & Configuration

PawDx is a client-side platform. To enable full functionality:

1. **Supabase**: 
   - Create a Supabase project.
   - Run the provided `schema.sql` in the SQL Editor.
   - Obtain your Supabase URL and Anon Key.
2. **OpenAI**:
   - Obtain an OpenAI API key with access to `gpt-4o-mini`.
3. **App Settings**:
   - Navigate to the **Dashboard** in PawDx.
   - Enter your Supabase and OpenAI keys in the configuration section.
   - Sign up/Login via the Supabase Auth system.
